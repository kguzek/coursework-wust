# Diagram klas analitycznych — System E-Wybory

Źródło diagramu PlantUML: [`e-wybory-klasy-analityczne.puml`](./e-wybory-klasy-analityczne.puml)

Diagram obejmuje klasy uczestniczące w realizacji przypadków użycia UC1-UC8:

- klasy sterujące `<<control>>`
- klasy encyjne `<<entity>>`
- bez klas `<<boundary>>` reprezentujących API

Renderowanie do PNG:

```bash
plantuml e-wybory-klasy-analityczne.puml
```

Renderowanie do SVG:

```bash
plantuml -tsvg e-wybory-klasy-analityczne.puml
```
